## 2. 技术描述

- **前端**：React@18 + TypeScript + Vite
- **可视化库**：D3.js@7（力导向图）
- **算法执行**：Web Worker（独立线程执行BFS）
- **样式**：TailwindCSS@3
- **初始化工具**：Vite

## 4. 数据模型

### 4.1 图数据结构

```typescript
interface Node {
  id: string;
  label?: string;
}

interface Edge {
  source: string;
  target: string;
}

interface GraphData {
  nodes: Node[];
  edges: Edge[];
}
```

### 4.2 算法状态数据结构

```typescript
interface BFSState {
  current: string | null;
  visited: Set<string>;
  queue: string[];
  isComplete: boolean;
  step: number;
}
```
