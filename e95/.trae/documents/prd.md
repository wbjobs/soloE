## 1. 产品概述

量子态模拟器是一个交互式量子计算可视化工具，允许用户通过拖拽方式搭建量子电路，实时观察量子态演化。目标用户为量子计算初学者、教育工作者和研究人员。

产品价值：通过直观的可视化界面降低量子计算学习门槛，帮助用户理解量子门操作和量子态演化原理。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 使用所有量子门搭建电路、运行模拟、查看态矢量 |

### 2.2 功能模块

1. **电路设计面板**：量子门选择、拖拽放置、线路连接、电路清除
2. **模拟器控制**：运行电路、单步执行、重置状态
3. **态矢量可视化**：实部/虚部柱状图、概率分布显示
4. **测量功能**：量子比特测量、测量结果统计

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 量子门工具箱 | 展示 H, X, CNOT, Y, S, T 等量子门，支持拖拽到电路 |
| 主页面 | 电路画布 | 显示量子比特线路，支持门的放置、移动和删除 |
| 主页面 | 控制面板 | 运行按钮、重置按钮、量子比特数量选择（最多 12） |
| 主页面 | 可视化区域 | Plotly 柱状图展示态矢量的实部和虚部 |
| 主页面 | 测量结果 | 显示各量子态的测量概率和最近一次测量结果 |

## 3. 核心流程

用户从工具箱拖拽量子门到对应量子比特的时间步位置，构建量子电路后点击运行，后端执行模拟计算并返回态矢量，前端渲染柱状图展示结果。用户可对特定量子比特进行测量，观察波函数坍缩效果。

```mermaid
flowchart LR
    A["拖拽量子门到电路"] --> B["调整门位置和参数"]
    B --> C["点击运行按钮"]
    C --> D["后端执行量子模拟"]
    D --> E["返回态矢量数据"]
    E --> F["前端渲染柱状图"]
    F --> G["可选：测量量子比特"]
    G --> H["显示测量结果和坍缩后状态"]
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝科技感 (#0a192f)，强调量子计算的科学属性
- 辅助色：量子蓝 (#64ffda)、紫色 (#bd93f9)、橙色 (#ffb86c)
- 背景：深色渐变背景，配合微妙的网格纹理
- 按钮：圆角矩形，悬浮时有发光效果
- 字体：JetBrains Mono 作为代码和数字显示，Inter 作为界面文字
- 图标：简洁线性图标，量子门用符号表示 (H, X, ⊕, Y, S, T)

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 主页面 | 顶部导航 | 标题、Logo、GitHub 链接 |
| 主页面 | 左侧工具箱 | 量子门卡片列表，拖拽手柄 |
| 主页面 | 中央电路区 | 水平线表示量子比特，竖线分隔时间步，门用彩色方块表示 |
| 主页面 | 右侧控制面板 | 运行/重置按钮、量子比特数滑块 |
| 主页面 | 底部图表区 | Plotly 双柱状图（实部蓝色、虚部橙色） |
| 主页面 | 测量结果区 | 概率表格，最近测量结果高亮显示 |

### 4.3 响应式

桌面端优先布局，在窄屏设备上自动将工具箱改为顶部横向排列，图表区域自适应宽度。

### 4.4 交互细节

- 量子门拖拽时显示半透明预览
- 电路悬停显示门的矩阵信息
- 态矢量更新时有平滑过渡动画
- 测量时概率柱状图有闪烁效果
