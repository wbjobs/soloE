# 用户使用指南

## 快速开始

### 1. 注册账号
1. 访问官网 www.example.com
2. 点击"免费试用"按钮
3. 填写邮箱和密码
4. 验证邮箱完成注册

### 2. 首次登录
1. 使用注册的邮箱和密码登录
2. 完成初始设置向导
3. 上传您的第一批文档
4. 开始使用系统

### 3. 上传文档
1. 点击左侧导航栏的"上传"按钮
2. 选择要上传的文件
3. 等待上传完成
4. 系统将自动处理和索引文档

## 高级功能

### 智能标签
系统会自动为您的文档生成智能标签。您也可以手动添加或修改标签：
1. 打开文档详情页
2. 点击标签区域
3. 输入新标签或删除现有标签
4. 保存更改

### 文档问答
使用文档问答功能，您可以用自然语言提问，系统会根据您的文档内容给出答案：
1. 在搜索框中输入问题
2. 按回车键或点击搜索按钮
3. 查看AI生成的回答
4. 点击"查看来源"查看相关文档片段

### 批量操作
对多个文档进行批量操作：
1. 勾选要操作的文档
2. 点击"批量操作"按钮
3. 选择操作类型（移动、删除、导出等）
4. 确认操作

## 常见问题

### Q: 支持哪些文件格式？
A: 系统支持超过50种文件格式，包括但不限于：
- 文档：PDF, DOC, DOCX, TXT, MD
- 表格：XLS, XLSX, CSV
- 演示：PPT, PPTX
- 图片：JPG, PNG, GIF

### Q: 文档上传后多久可以搜索到？
A: 通常情况下，文档上传后1-5分钟内即可被搜索到。具体时间取决于文档大小和系统负载。

### Q: 我的数据安全吗？
A: 我们非常重视数据安全：
- 所有数据传输都使用SSL加密
- 数据存储采用AES-256加密
- 定期备份数据
- 符合GDPR和等保三级标准

### Q: 如何联系客服？
A: 您可以通过以下方式联系我们：
- 在线客服：网站右下角的聊天按钮
- 邮件：support@example.com
- 电话：400-123-4567（工作日 9:00-18:00）
