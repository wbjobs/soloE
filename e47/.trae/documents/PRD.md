# 人像抠图工具 - 产品需求文档

## 1. 产品概述

基于浏览器的实时人像抠图工具，使用ONNX Runtime Web运行预训练模型，支持摄像头实时处理和图片上传两种模式。

- **核心价值**: 提供高性能、高质量的浏览器端人像分割和背景替换功能
- **目标用户**: 需要快速制作证件照、视频会议背景替换、创意图像处理的用户

## 2. 核心功能

### 2.2 功能模块

1. **主界面**: 视频/图片预览区域、控制面板、功能切换
2. **输入模块**: 摄像头实时采集、图片文件上传
3. **抠图处理**: ONNX模型推理、WebGL后处理
4. **背景替换**: 纯色背景、模糊背景、自定义背景图
5. **导出功能**: 处理结果下载

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 预览区域 | 640x480分辨率实时预览，支持原图和处理后对比 |
| 主页面 | 输入控制 | 摄像头开关、切换前后摄像头、图片上传 |
| 主页面 | 背景设置 | 纯色选择器、模糊滑块、背景图上传 |
| 主页面 | 后处理控制 | 边缘羽化、腐蚀膨胀强度调节 |
| 主页面 | 导出功能 | 下载处理后的图片 |

## 3. 核心流程

```mermaid
flowchart LR
    A["用户进入页面"] --> B{"选择输入方式"}
    B -->|摄像头| C["请求摄像头权限"]
    B -->|上传图片| D["加载图片文件"]
    C --> E["开始视频采集"]
    D --> E
    E --> F["Web Worker加载ONNX模型"]
    F --> G["实时推理生成Alpha遮罩"]

    J --> K{"用户导出图片"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深灰色系，突出科技感和专业性
- **辅助色**: 亮蓝色作为交互元素高亮
- **按钮风格**: 圆角矩形，微妙的悬停动画
- **字体**: 使用现代无衬线字体，层级清晰
- **布局**: 左右分栏布局，左侧预览，右侧控制面板
- **图标风格**: 使用线性图标，简洁现代

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 主页面 | 预览区域 | 居中显示，边框圆角，阴影效果 |
| 主页面 | 控制面板 | 卡片式设计，分组折叠 |
| 主页面 | 控制按钮 | 分组排列，状态指示清晰 |
| 主页面 | 加载状态 | 骨架屏+进度条 |

### 4.3 响应式

- **桌面端优先**: 左右分栏布局
- **平板适配**: 上下布局
- **移动端**: 单列布局，触控优化
